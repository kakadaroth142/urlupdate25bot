# urlupdate25bot рҹҺҘ

Bot Telegram бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ¶бһүбһңбһ·бһҠбҹҒбһўбһјбһ–бһёбһӣбҹҒбһҒбһҹбһјбһ…бһ“бһ¶бһҖбһҡбһ•бҹ’бһҹбҹҒбһ„бҹ—

## бһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹ вӯҗ

- вң… бһ‘бһ¶бһүбһңбһ·бһҠбҹҒбһўбһјбһҠбҹ„бһҷ URL
- вң… бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ Resolution (480p, 720p, 1080p)
- вң… бһҜбһҖбһҹбһ¶бһҡбһ”бһ„бҹ’бһҡбһҪбһҳ
- вң… бһ…бһӣбһ“бһ¶бһӣбһҳбҹ’бһўбһ·бһҸ Khmer

## бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡ рҹ“Ӣ

- Python 3.8+
- Telegram Bot Token
- бһҜбһҖбһҹбһ¶бһҡбһӣбһҳбҹ’бһўбһ·бһҸбһҖбҹ’бһ“бһ»бһ„ `requirements.txt`

## бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ рҹҡҖ

### 1. Clone бһҡбһёбһ”бһј:
```bash
git clone https://github.com/kakadaroth142/urlupdate25bot.git
cd urlupdate25bot
```

### 2. бһҠбҹҶбһЎбһҫбһ„ dependencies:
```bash
pip install -r requirements.txt
```

### 3. бһ”бһ„бҹ’бһҖбһҫбһҸ `config.py`:
```python
BOT_TOKEN = "your_telegram_bot_token_here"
DOWNLOAD_FOLDER = "downloads"
```

бһ¬бһӣбһҳбҹ’бһўбһ·бһҸбһҜбһҖбһҹбһ¶бһҡ `config.py` бһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһҸбһӣбҹӢбһұбҹ’бһҷ бһ бһҫбһҷбһҠбһ¶бһҖбҹӢбһӣбҹҒбһҒбһҖбһјбһҠ Bot Token бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

### 4. бһҡбһҸбҹӢ Bot:
```bash
python3 main.py
```

## бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ рҹ’¬

- `/start` - бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ Bot
- `/help` - бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһӣбһҳбҹ’бһўбһ·бһҸбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ
- бһңбһ¶бһ„ URL бһңбһ·бһҠбҹҒбһўбһј - бһ‘бһ¶бһүбһңбһ·бһҠбҹҒбһўбһј

## бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣ рҹ”§

```
1. бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ•бҹ’бһүбһҫ URL бһңбһ·бһҠбҹҒбһўбһј
2. бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ Resolution бһҠбҹӮбһӣбһ…бһ„бҹӢбһ”бһ¶бһ“
3. бһҡбһ„бҹӢбһ…бһ¶бҹҶбһҖбһ¶бһҡбһ‘бһ¶бһүбһңбһ·бһҠбҹҒбһўбһј
```
